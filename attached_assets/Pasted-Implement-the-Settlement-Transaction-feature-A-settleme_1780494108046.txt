Implement the Settlement Transaction feature.

A settlement records a financial event that is not a buy or sell — dividends, fees, taxes, zakat, purification payments, or corrections. Settlements affect the investor’s real return but never change share quantity, cost basis, avg cost, or FIFO lots.

settlement_amount: float = 0.0 (positive = income, negative = expense)
settlement_category: str = “” (from the categories below)
settlement_currency: str = “” (ISO currency code)
