Watchlist search behavior:
- Add an autocomplete search field in the Watchlist page.
- Search by Yahoo ticker, Sahmk ticker, local ticker, canonical asset name, platform asset name, or partial name.
- Display result as:
  Canonical Asset Name — Market — Currency
  Yahoo: ticker if available
  Sahmk: ticker if available
  Primary Source: source name

Selection behavior:
- When selected, add the asset only to the Watchlist.
- Store watchlist_asset_id in the watchlist item.
- Do not create or modify holding asset_id.
- Do not create transactions.
- Do not affect actual portfolio positions.

Price lookup priority inside Watchlist only:
- For Saudi assets:
  1. Try Sahmk ticker first.
  2. If unavailable, fallback to Yahoo ticker.
  3. If unavailable, show manual price required.

- For US/global assets:
  1. Try Yahoo ticker first.
  2. If unavailable, check other linked source.
  3. If unavailable, show manual price required.

- Link multiple platform tickers to one watchlist asset
- Mark one ticker source as primary
- Activate/deactivate mappings
